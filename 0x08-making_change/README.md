## About <a name="about"></a>
This project is part of the curriculum of the ALX Software Engineering program. The main objective of this project is to understand and implement Dynamic Programming. It was done in preparation for future job interviews.

## Learning Objectives <a name="learning-objectives"></a>
- To be able to solve dynamic programming interview questions and apply dynamic programming in future projects.

## Requirements <a name="requirements"></a>
- Allowed editors: `vi`, `vim`, `emacs`
- All your files will be interpreted/compiled on Ubuntu 20.04 LTS using `python3` (version 3.4.3)
- All your files should end with a new line
- The first line of all your files should be exactly `#!/usr/bin/python3`
- A README.md file, at the root of the folder of the project, is mandatory
- Your code should use the `PEP8`  style (version 1.7.x)
- All your files must be executable

## Getting Started <a name="getting-started"></a>
To get started with this project, do the following:

1. Clone this repository: `git clone https://github.com/Pritchad25/alx-interview.git`
2. Access the project directory: `cd 0x08-making_change`
3. Run the program: `./main.py`
